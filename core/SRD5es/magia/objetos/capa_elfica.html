Capa élfica

Objeto maravilloso, poco común (requiere sintonización)

Mientras llevas esta capa con la capucha puesta, las pruebas de Sabiduría (Percepción) que se hagan para verte tienen desventaja y tú tienes ventaja en las pruebas de Destreza (Sigilo) que hagas para esconderte, ya que la capa cambia de color para camuflarte. Ponerte o quitarte la capucha requiere una acción.
