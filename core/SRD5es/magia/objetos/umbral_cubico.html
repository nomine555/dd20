Umbral cúbico

Objeto maravilloso, legendario

Este cubo tiene una longitud de 3 pulgadas e irradia una energía mágica palpable. Cada una de sus caras está asociada a un plano de existencia diferente, uno de los cuales es el Plano Material. Las otras caras están conectadas con los planos que determine el director de juego.

Puedes usar una acción para presionar una de las caras del cubo para lanzar el conjuro Umbral, lo que abre un portal al plano con el que está conectada dicha cara. De manera alternativa, si usas una acción para presionar una cara dos veces, puedes lanzar el conjuro Desplazamiento de plano (salvación CD 17) con el cubo y transportar a los objetivos al plano conectado con dicha cara.

El cubo tiene 3 cargas y cada uso gasta 1. El cubo recupera 1d3 cargas cada amanecer.
