Cetro inamovible

Cetro, poco común

Este cetro plano de hierro tiene un botón en uno de sus extremos. Puedes usar una acción para presionar el botón, lo cual hace que el centro se fije mágicamente a un lugar. Hasta que tú u otra criatura uséis una acción para apretar el botón de nuevo, el cetro no se mueve, incluso si desafía a la gravedad. El cetro puede aguantar hasta 8000 libras de peso, pero más hace que el cetro se desactive y se caiga. Una criatura puede usar una acción para hacer una prueba de Fuerza CD 30; si tiene éxito, mueve el cetro fijo hasta 10 pies.
