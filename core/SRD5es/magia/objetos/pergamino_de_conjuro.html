Pergamino de conjuro

Pergamino, varía

Un pergamino de conjuro contiene las palabras de un único conjuro, escrito en un código místico. Si el conjuro está en la lista de conjuros de tu clase, puedes leer el pergamino y lanzar el conjuro sin necesidad de ningún componente material. Si no, el pergamino es inteligible. Lanzar este conjuro leyendo el pergamino requiere el tiempo habitual de lanzamiento. Una vez se lanza, las palabras se desvanecen y el pergamino se convierte en polvo. Si se interrumpe el lanzamiento, el pergamino no se pierde.

Si el conjuro está en tu lista de conjuros de clase pero es de un nivel superior al que puedes lanzar normalmente, debes hacer una prueba de característica de tu característica para lanzar conjuros para determinar si puedes lanzarlo con éxito. La CD es igual a 10 + el nivel del conjuro. Si fallas, el conjuro desaparece del pergamino sin surtir efecto.

El nivel del conjuro del pergamino determina la CD de la tirada de salvación del conjuro y su bonificador al ataque, así como la rareza del pergamino, como se indica en la tabla de pergaminos de conjuro.

Un conjuro de mago de un pergamino de conjuro se puede copiar igual que los conjuros de un libro de conjuros. Al hacerlo, el copista debe hacer una prueba de Inteligencia (Arcanos) con una CD igual a 10 + el nivel del conjuro. Si la supera, el conjuro se copia con éxito. Tanto si tiene éxito como si no, el pergamino de conjuro se destruye.
