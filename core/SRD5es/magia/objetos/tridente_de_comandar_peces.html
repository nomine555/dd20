Tridente de comandar peces

Arma (tridente), poco común (requiere sintonización)

Este tridente es un arma mágica. Tiene 3 cargas. Mientras lo llevas, puedes usar una acción y gastar 1 carga para lanzar con él Dominar bestia (salvación CD 15) sobre una bestia que tenga velocidad nadando innata. El tridente recupera 1d3 cargas cada amanecer.
