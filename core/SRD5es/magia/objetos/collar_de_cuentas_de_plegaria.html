Collar de cuentas de plegaria

Objeto maravilloso, raro (requiere sintonización por parte de un clérigo, druida o paladín)

Este collar tiene 1d4+2 cuentas mágicas de aguamarina, perla negra o topacio. También tiene varias cuentas no mágicas hechas de piedras como ámbar, heliotropo, cuarzo citrino, coral, jade o cuarzo. Si se saca una cuenta mágica del collar, dicha cuenta pierde su magia.

Existen seis tipos de cuentas mágicas. El director de juego decide o determina al azar el tipo de cada una de las del collar, que pueden del mismo tipo. Para usar una cuenta, debes llevar el collar. Cada cuenta contiene un conjuro que puedes lanzar como acción adicional (usa tu CD de salvación de conjuro si es necesario hacer una tirada de salvación). Una vez se lanza el conjuro de una cuenta mágica, dicha cuenta no se puede volver a usar hasta el siguiente amanecer.
